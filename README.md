# About Me Page

A simple about me page hosted on GitHub Pages.

## Setup Instructions

1. Replace `profile.png` with your own PNG image
2. Update the content in `index.html` with your information
3. Change the background color by modifying the `--background-color` variable in `style.css`
4. Add your social media links in the footer section

## Customization

- Change background color: Edit the hex code in `:root { --background-color: #your-color; }`
- Update profile picture: Replace `profile.png` with your image
- Modify text content in `index.html`
